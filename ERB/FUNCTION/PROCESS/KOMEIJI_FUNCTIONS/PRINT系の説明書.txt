◎뭐야 이것?
  ·내가구상를 쓰려고했다등 어느새 함수만 증가하고 있었다
  ·그 중에서도 PRINT에 관한 것을 1개소에 모아 보았다
  ·완성도는 짐작

◎기능 일람
  사용할 수 있는 함수                  파일명      개요
  --□PRINT계□---------------------------------------------------------------------------------------------------------------------------------------
  @FPRINT(|D)(|L|W)           FPRINT계. ERB    PRINT 한다. 대개의 PRINT계 함수는 인수로 색변경할 수 있거나 표시 위치 변경할 수 있거나 한다
  @BIGPRINT(L|W)              BIGPRINT계. ERB  크게 PRINT 한다. BIGPRINT계는 SETFONT를 무시한다
  @SMALLPRINT(|L|W)           BIGPRINT계. ERB  작게 PRINT 한다. SMALLPRINT계는 SETFONT를 무시한다
  @HTMLPRINT(|D)(L|W)         HTML계. ERB      PRINT 한다. HTML_PRINT에, 억지로 SETCOLOR, FONTSTYLE, SETFONT, ALIGNMENT를 적용시켰다느낌
  @HTMLPRINTSINGLE(|D)        HTML계. ERB      상기의 PRINTSINGLE판
  @HTMLBIGPRINT(L|W)          HTML계. ERB      크게 PRINT 하는 HTML판. 도중에 폰트를 바꾸는 것은 법도
  @HTMLSMALLPRINT(L|W)        HTML계. ERB      작게 PRINT 하는 HTML판. 도중에 폰트를 바꾸는 것은 법도
  @PRINT_RAINBOW(|L|W)        RAINBOW계. ERB   무지개색으로 PRINT 한다. RAINBOW계는 과연 색설정이나 표시 위치의 인수를 가지지 않는다
  @BIGPRINT_RAINBOW(L|W)      RAINBOW계. ERB   크고 무지개색으로 PRINT 한다
  @SMALLPRINT_RAINBOW(|L|W)   RAINBOW계. ERB   작고 무지개색으로 PRINT 한다
  @PRINT_RAINBOW2(|L|W)       RAINBOW계. ERB   무지개색으로 PRINT 한다. 색이변화 하는애니메─숀 첨부. 행의 도중에 사용하면 위험       ※에 째─실마리응확장
  @BIGPRINT_RAINBOW2(L|W)     RAINBOW계. ERB   크고 무지개색으로 PRINT 한다. 색이변화 하는애니메─숀 첨부. 행의 도중에 사용하면 위험 ※에 째─실마리응확장
  @SMALLPRINT_RAINBOW2(|L|W)  RAINBOW계. ERB   작고 무지개색으로 PRINT 한다. 색이변화 하는애니메─숀 첨부. 행의 도중에 사용하면 위험 ※에 째─실마리응확장
  --□그 이외□--------------------------------------------------------------------------------------------------------------------------------------
  @ALIGNMENTS                 ALIGNMENTS.ERB  ALIGNMENTS 명령에 캐릭터 라인 변수 건네줄 수 있는 같게 함수를 씌운 것 뿐. PRINT계 함수 만들기에 편리. 다른 장소에서도 사용할 수 있다
  @GET_RAINBOW                RAINBOW계. ERB   RAINBOW의 색을 정의하고 있는 식 안의 함수. 비슷한 함수 만들거나 비슷한 색을 갖고 싶을 때에 사용한다
  @HTMLAUTOBUTTON             HTML계. ERB      버튼의 유효·무효상태에 응해<button>태그를 부가하는 식 안의 함수
  @HTMLBUTTON                 HTML계. ERB      <button>태그를 부가하는 식 안의 함수
  @HTMLFONT                   HTML계. ERB      <font>태그를 부가하는 식 안의 함수
  @HTMLNOBR                   HTML계. ERB      <nobr>태그를 부가하는 식 안의 함수
  @HTMLNONBUTTON              HTML계. ERB      <nonbutton>태그를 부가하는 식 안의 함수
  @HTMLP                      HTML계. ERB      <p>태그를 부가하는 식 안의 함수
  @HTMLSTYLE                  HTML계. ERB      <b><i><s><u>태그를 부가하는 식 안의 함수. FONTSTYLE 명령인 것 같지만, 성질상 Style를 되돌리는 것은 할 수 없다
  @CHKFACE                    HTML계. ERB      HTML 사양에 근거하는<font>태그의 face 속성의 복수 폰트 지정의 대체 기능인 식 안의 함수. 다른 장소에서도 사용할 수 있다
  @HTMLCOLOR                  HTML계. ERB      <font>태그에 색치를 건네줄 때에 적절한 포맷을 실시하는 식 안의 함수. 그다지 직접 사용하지 않는다
  ※D, L, W등의 의미는 PRINT 명령과 똑같습니다@FPRINT도 있으면, @FPRINTDL도 있습니다
  ※인수에 FORM 캐릭터 라인을 건네줄 때는""의 전에@를 붙여 주세요   예) @" 「%MASTERNAME%, 기상의시간입니다」"
  ※자세한 것은 각 파일에 기재되어 있는 각 함수의 코멘트를 확인해 주세요


◎사용법
  ※구상로 사용하고 싶은 경우는 이것을 바리안트측 거두어들여 받을 수 없는가, 우선 부탁해 보는 것이 좋을지도
  01. 함수명이나 파일명 따위의""의 부분을 적절히 치환합니다. 이것은 단지 함수명의 중복 피하기인 것으로…
     바리안트라면"", 구상라면"KXX_"(XX=그 캐릭터의번호)라든지로 하면 좋은 느낌에
     그대로 던져 넣는 것은 여러가지 의미로 피하는 편이 좋다. 치환하고 나서 배포했다 (분)편이 좋았던 생각도 들지만… 개발 환경적인 형편이라든지 다양하구나(거짓말
  02. 바리안트라면 ERB 폴더의 한쪽 구석, 구상라면 그근처의 어디엔가 적당하게 던져 넣습니다
  03. 애니메─숀 하는 PRINT계에 째─실마리응과 제휴하기 때문에, 없는 경우는 도입 추천
     라고 해도구상등으로는 어려울지도 모르기 때문에, RAINBOW계. ERB를 개조해 해당 부분을 깎는다고 하는손도
     (해당 부분→IF ANIME_CONFIG("SCROLLTEXT")와 거기에 대응하는 ENDIF를 comment out)
     (ver1. 01의 경우, 177, 194, 257, 278행째를 각각 코멘트화하기만 하면 되는 간단한 일입니다)
  04. 이것으로 함수를 사용할 수 있습니다. 해냈군요!
  ※필요인 부분만큼 잘라 사용해도 되는거야(BIGPRINT계+ALIGNMENTS만이라든지, HTML계이지만인가, RAINBOW계만이라든지)
